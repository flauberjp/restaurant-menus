Iterative and incremental

Breakdown your development in small phases, involve stakeholdes, keep development feature by feature, and use this checklist to guide the actions to be done on each phase.

Checklist

  1. Mock-ups: For every page, including their URL
  2. Routing: Define all URL for your application, this way you can navigate to all URL even if the pages are not yet created
  3. Templates & Forms: On top of others to iterations, create all templates and forms
  4. CRUD Functionality: Add all operations are working properly with data being retrieved from the database
  5. API Endpoints: Retrieve data of the menus in JSON format
  6. Styling & Message Flashing: Add styling, images and messages
